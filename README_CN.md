# 中文
贝爷数据包1.20.5+，本数据包复刻老版 贝爷模组

安装&游玩方法
下载后的压缩包无需解压放到.minecraft\saves<world name>\datapacks文件夹下，进入世界后输入 /reload 即可。

推荐使用原版服务端或者spigot端进行游玩，使用服务端游玩需要修改function的权限等级，在server.properties文件中

function-permission-level=3
